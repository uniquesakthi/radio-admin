## ğŸš€ Quick Start

### Prerequisites

- A Firebase project (free tier works perfectly)
- A web browser
- Basic text editor

### Setup Instructions

#### 1. Create Firebase Project

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Click "Add Project" or select existing project
3. Follow the setup wizard

#### 2. Enable Firebase Authentication

1. In Firebase Console, go to **Authentication** â†’ **Sign-in method**
2. Enable **Email/Password** authentication
3. Click **Save**

#### 3. Create Firestore Database

1. In Firebase Console, go to **Firestore Database**
2. Click **Create Database**
3. Choose **Start in production mode** (we'll set rules next)
4. Select your preferred location
5. Click **Enable**

#### 4. Set Firestore Security Rules

Go to **Firestore Database** â†’ **Rules** and paste:

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Only authenticated users can read/write
    match /channels/{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
```

Click **Publish** to save the rules.

#### 5. Get Firebase Configuration

1. In Firebase Console, go to **Project Settings** (gear icon)
2. Scroll down to "Your apps"
3. Click the **Web** icon (`</>`)
4. Register your app with a nickname (e.g., "Radio Admin")
5. Copy the `firebaseConfig` object

#### 6. Configure the Application

1. Open `app.js` in your text editor
2. Find the `firebaseConfig` object at the top:

```javascript
const firebaseConfig = {
    apiKey: "YOUR_API_KEY",
    authDomain: "YOUR_AUTH_DOMAIN",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_STORAGE_BUCKET",
    messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
    appId: "YOUR_APP_ID"
};
```

3. Replace with your actual Firebase config values
4. Save the file

#### 7. Create Admin User

1. In Firebase Console, go to **Authentication** â†’ **Users**
2. Click **Add User**
3. Enter email and password for your admin account
4. Click **Add User**

#### 8. Run the Application

**Option A: Using Python (if installed)**
```bash
cd c:\Users\ADMIN\Desktop\radio_admin
python -m http.server 8000
```
Then open: http://localhost:8000

**Option B: Using Node.js (if installed)**
```bash

```

**Option C: Using VS Code Live Server**
1. Install "Live Server" extension in VS Code

**Option D: Direct File Access**
Simply double-click `index.html` (may have CORS issues with Firebase)

## ğŸ“– Usage Guide

## ğŸ—‚ï¸ Data Structure

Each channel document in Firestore has the following structure:

```javascript
{
  title: "Classic Rock FM",           // Channel name
  category: "Rock",                   // Category for grouping
  url: "https://stream.example.com",  // Stream URL

  createdAt: "2025-11-25T...",       // Auto-generated

```

The document ID is auto-generated by Firestore.
